PROCEDIMENTO:
1º Instalar o XAMPP com o MySQL e Apache
2º Instalar o Node.js
3º Importar a base de dados "er.sql" no phpmyadmin
4º Aceder à diretoria do projeto (Projeto ER) no terminal e correr o comando "npm start"
5º Divertir-se :D

NOTA: Assegure-se que o XAMPP está aberto (Apache e MySQL online) ao executar o passo 4.
