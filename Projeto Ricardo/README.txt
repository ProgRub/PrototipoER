Para isso tens de seguir os seguintes passos:

- Instala a versão do Xampp mais atualizada (https://www.apachefriends.org/download.html)
    . O Xampp é uma aplicação para teres acesso a um servidor local.
    . IMPORTANTE: É importante teres configurado o Xampp de forma correta, verifica se a tua porta é a 3306.
- Depois de teres instalado o Xampp, clica em "Start" no Apache e MySQL (dentro da aplicação do Xampp)
- Abre no teu browser o seguinte endereço: http://localhost/phpmyadmin/index.php e importa o ficheiro .sql que disponbilizamos para teres acesso a nossa Base de Dados.
- Após teres a Base de Dados no teu sistema, dirige-te a pasta Grupo 1 - ER/Electron e lá encontraras um ficheiro executavel (.exe) com o nome "CodeMent - Grupo 1". 
    . Este é um ficheiro executavel da nossa aplicação, faz click e espera até a aplicação aparecer completamente carregada.
